Sistema de Información para Componentes

Editor de Componentes

Dibujar el Componente

El componente puede dibujarse utilizando cualquier elemento presente en la barra de dibujo como pueden ser:

þ     Líneas y polilíneas.

þ     Rectángulos.

þ     Elipses.

þ     Curvas.

Insertar los conectores

Los conectores que pueden tener los componentes pueden ser de dos tipos:

þ     Tipo conector de componente o J/P (jack / plug).

þ     Tipo empalme de cable.

Insertar las etiquetas

Las etiquetas corresponden a:

þ     Las etiquetas de identificación del componente.

þ     Las etiquetas de identificación de los diferentes pines.

Situar el contorno del componente

El contorno del componente debe situarse de tal manera que contenga todos los elementos del componente con el tamaño mínimo.

Nota        Se pueden dejar fuera del contorno los puntos de conexión de los diferentes conectores.
