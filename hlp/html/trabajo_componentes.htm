Trabajo con Componentes

El trabajo con componentes permite poblar cada hoja del diseño, posteriormente, es necesario interconectar todos los componentes mediante cables, buses y puertos.

Las principales tareas en el trabajo con componentes son las siguientes:

þ     Insertar Componentes

þ     Mover / Editar Componentes

þ     Eliminar Componentes

Insertar Componentes

Pulsar el botón Insertar Componente de la barra de herramientas insertar o el comando Componente del menú Insertar.

Mover / Editar Componentes

þ     Para mover un componente situar el puntero del ratón sobre él, pulsar sin soltar el botón izquierdo del ratón sin soltar, arrastrar hasta la nueva posición y soltar.

Si el componente tuviera cables conectados en algunos de sus conectores, estos se alargarían de forma proporcional, para seguir conectados.

þ     Para editar las propiedades de un componte hacer doble clic con el botón izquierdo del ratón sobre él, o bien hacer clic con el botón derecho y seleccionar Propiedades del menú emergente.

þ     Para editar el componente en el Sistema de Información de Componentes: seleccionar el componente con el botón izquierdo y pulsar Editar Componente en la barra de herramientas insertar, o bien hacer clic en Editar Componente en SIC en el menú emergente del botón derecho del ratón.

Eliminar Componentes

Situar el puntero del ratón sobre el componente y hacer clic con el botón izquierdo para que el componente quede seleccionado.

Para eliminar el componente, pulsar la tecla Supr o seleccionar Eliminar del menú Editar.

También es posible elegir Eliminar del menú emergente que aparece cuando se hace clic con el botón derecho del ratón sobre el componente.
