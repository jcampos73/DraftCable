Instalación de Estaciones de Trabajo

1. Acceder a través del entorno de red al servidor y a la carpeta compartida de Draft Cable.

2. Ejecutar la instalación de Estación de Trabajo.

3. Después de raelizar la instalación, se nos pregunta si queremos ejecutar el programa.

4. Se nos pregunta una contraseña de administrador válida para poder acceder al sistema.
