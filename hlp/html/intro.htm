Draft Cable CAD para Telecomunicaciones

Draft Cable Sistema CAD ('Computer Aided Design') para Telecomunicaciones permite al ingeniero un herramienta que le acompaña en todas las fases del diseño de un sistema de telecomunicaciones complejo:

þ     Diseño de todos los esquematicos, con separación por hojas.

þ     Generación de los alzados de los rack de manera semi-automática.

þ     Generación de las hojas de cableado para el personal de montajes.

Nota para nuevos usuarios

Para empezar a usar el programa de la manera más rápida posible, conviene dirigirse a la sección Como hacer...

Nota para el administrador del sistema

Dado que Draft Cable es un sistema multi-usuario preparado para entornos cliente-servidor, conviene que los administradores de sistemas se dirijan a la sección Documentación de Administrador.
