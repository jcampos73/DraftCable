Recomendaciones Hardware

En esta sección se incluyen comentarios accerca de los requisitos hardware para que el sistema funcione de una manera óptima.

Hardware de Servidor

El servidor conviene que cuente con un array de discos duros interno (o externo, tipo SAN o similar) de tipo RAID5, quizá con un disco 'Hot-Spare' para garantizar la seguridad de los datos

Por otro lado, sería conveniente instalar un sistema de backup automático en DVD de tipo Robot Grabador de DVD's, con torre de discos en blanco e impresora de etiquetas o con librería tipo Jukebox

En cuanto al tipo de red, se recomienda una red a 1 Gb para maximizar el rendimiento del sistema.
