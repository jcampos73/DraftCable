generación de hojas de cableado

Formato Microsoft Word 2000

Este formato es el que permite más control sobre el aspecto final de la documentación. Sin embargo, puede ser más lento según sea el número de hojas que se vayan a generar y la potencia de la estación de trabajo.

Este formato es adecuado para producir la documentación de salida según el formato corporativo de la empresa.

Formato HTML

Este es un formato mucho más rápido en su generación. Sin embargo, el usuario no tiene tanto control sobre el aspecto final.

Es adecuado para generación no definitiva o de ejemplo para inserción en otros dicumentos, ya que el formato HTML es fácilmente importable en otros documentos.

También es útil este formato para la publicación de documentación en una intarnet corporativa.
