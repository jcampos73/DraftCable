Comando Abrir (menú Archivo)

Use este comando para abrir un documento que ya existe en una nueva ventana. Puede abrir varios documentos a la vez. Use el menú Ventana para alternar entre los diversos documentos abiertos. Consulte el comando Ventana > 1, 2, ....

Puede crear nuevos documentos con el comando Nuevo.

Método abreviado

Barra de herramientas:

Teclas:CTRL+A
