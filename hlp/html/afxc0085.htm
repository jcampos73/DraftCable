Comando Cerrar (menús de control)

Use este comando para cerrar la venta o cuadro de diálogo activos.

El hacer doble clic en una casilla de Control de menú cumple la misma función que el comando Cerrar.

Nota: Si tiene múltiples ventanas abiertas para un mismo documento, el comando Cerrar del menú Control del documento cierra sólo una ventana a la vez. Puede cerrar todas las ventanas de una vez mediante el comando Cerrar del menú Archivo.

Método abreviado

Teclas: CTRL+F4 cierra las ventanas de documento

ALT+F4 cierra las ventanas o cuadros de diálogo de <<YourType>>
