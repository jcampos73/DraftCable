Comando Cerrar (menú Archivo)

Use este comando para cerrar todas las ventanas que contienen el documento abierto. Draft Cable le sugiere que guarde los cambios que se han hecho al documento antes de cerrarlo. Si cierra un documento sin guardarlo, se perderán todos los cambios introducidos desde la última vez que se guardó. Antes de cerrar un documento sin título, Draft Cable muestra el cuadro de diálogo Guardar como y le sugiere que asigne nombre al documento y lo guarde.
