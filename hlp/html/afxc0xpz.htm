Comando Nuevo (menú Ventana)

Use este comando para abrir una nueva ventana que tenga los mismos contenidos que la ventana activa. Puede abrir múltiples ventanas de documento para ver distintas partes o presentaciones de un documento al mismo tiempo. Si cambia el contenido de una de las ventanas, las demás ventanas que contienen el mismo documento también cambiarán. Cuando se abre una nueva ventana, ésta se transforma en la ventana activa y aparece encima de las demás ventanas.
