Pasos de la Instalación

Requisitos Previos

Bases de datos

1. Instalar MS SQL Server en el servidor.

2. La contraseña del usuario administrador 'sa' habrá que introducirla durante la instalación de Draft Cable en el servidor.

Carpetas Compartidas

1. Crear una carpeta compartida en el servidor desde donde instalar las Estaciones de Trabajo.

Instalación

1. Ejecutar el programa de instalación en el servidor.

2. Especificar como lugar de instalación la carpeta compartida que se especificó en los requitos previos. Introducir el nombre completos de red en la forma \\nombre del servidor\carpeta. Donde nombre del servidor es el nombre del servidor en el Dominio o Grupo de Trabajo, no introducir la IP del servidor directamente.

3. Se instalan las bases de datos en el MS SQL Server. Es necesario introducir la contraseña de administrador para el usuario 'sa'.

4. Se instalan ahora los drivers de la llave hardware de servidor.

5. Introducir la llave hardware, USB o Paralelo.

6. El programa de instalación copia todos los ficheros de Draft Cable.

7. Finalmente se nos pide que introduzcamos la contraseña del administrador de Draft Cable.
