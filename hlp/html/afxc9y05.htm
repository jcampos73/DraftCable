Comando Guardar (menú Archivo)

Use este comando para guardar el documento activo con su nombre y directorio actuales. Cuando guarda un documento por primera vez, Draft Cable muestra el cuadro de diálogo Guardar como para que pueda asignar un nombre al documento. Si desea cambiar el nombre y el directorio de un documento que ya existe antes de guardarlo, elija el comando Guardar como.

Barra de herramientas:

Teclas:CTRL+G
