Arquitectura de Draftcable

Introducción

Draft Cable es un programa cliente-servidor en el que un servidor corporativo almacana todos las bibliotecas y componentes.

Los clientes generan esquemáticos, alzados de racks y cableados a partir de estos componentes.

De esta manera, la información se comparte en toda la organización.

Existe también la posibilidad de guardar los esquemáticos completos en la base de datos. De esta manera se puede organizar fácilmente un sistema de archivo para todos los planos y documentación egenerada.

Los usuarios con suficientes priviliegios pueden modificar los componentes de las bibliotecas en el sistema o incluso crear bibliotecas nuevas.
