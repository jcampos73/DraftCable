Comando Dividir (menú Ventana)

Use este comando para dividir la ventana activa en secciones. Luego puede usar el Mouse o las teclas de flecha del teclado para mover las barras de separación. Cuando haya terminado, presione el botón del Mouse o ENTRAR para que las barras de separación queden en su nueva posición. Al presionar ESCAPE las barras de separación vuelven a su posición original.
