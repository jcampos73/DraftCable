Comando Cortar (menú Editar)

Use este comando para borrar los datos actualmente seleccionados del documento y colocarlos en el portapapeles. Este comando no está disponible si no hay datos actualmente seleccionados.

Al usar el comando Cortar para colocar datos en el portapapeles, se reemplazan los datos que previamente estaban almacenados en el mismo.

Método abreviado

Barra de herramientas:

Teclas:CTRL+X
