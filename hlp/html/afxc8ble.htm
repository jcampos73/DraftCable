Comando Barra de herramientas (menú Ver)

Use este comando para mostrar y ocultar la barra de herramientas, que tiene botones para algunos de los comandos más usados de Draft Cable, como el comando Abrir del menú Archivo. Cuando se muestra la barra de herramientas, aparece una marca de verificación al lado del elemento de menú.

Consulte Barra de herramientas para obtener ayuda sobre cómo usar la barra de herramientas.
