Cuadro de diálogo Especificar Impresora

Las siguientes opciones permiten seleccionar la impresora de destino y su conexión.

Impresora

Seleccione la impresora que desea usar. Elija la impresora predeterminada o elija la opción Impresora específica y seleccione una de las impresoras actualmente instaladas que aparecen en el cuadro. Use el Panel de control de Windows para instalar impresoras y configurar puertos.

Orientación

Elija vertical u horizontal.

Tamaño de papel

Seleccione el tamaño de papel en que se imprimirá el documento.

Origen del papel

Algunas impresoras tienen múltiples bandejas para ofrecer diversos orígenes de papel. Especifique aquí la bandeja.

Opciones

Muestra un cuadro de diálogo en el que se pueden seleccionar opciones adicionales de impresión según el tipo de impresora que ha seleccionado.

Red...

Elija este botón para conectarse a un lugar de la red asignándole una nueva letra de unidad.
