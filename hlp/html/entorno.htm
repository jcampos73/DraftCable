Editor de Planos

El editor de planos es un editor de esquemáticos que permite el posicionamiento de componentes y la interconexión de los mismos mediante cables, buses y puertos.

Herramienta Editor de 'patch-panel'

El editor de 'path-panel' permite crear de una manera rápida estos elementos y realizar la interconexión de canalales de una manera rápida.

Después, el Generador de Documentación generará las correspondientes hojas de puentes a partir de cada 'patch-panel'.
