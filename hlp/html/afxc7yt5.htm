Comando Copiar (menú Editar)

Use este comando para copiar los datos seleccionados en el portapapeles. Este comando no está disponible si no hay datos actualmente seleccionados.

Al usar el comando Copiar para colocar datos en el portapapeles, se reemplazan los datos que previamente estaban almacenados en el mismo.

Método abreviado

Barra de herramientas:

Teclas:CTRL+C
