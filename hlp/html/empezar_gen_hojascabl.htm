Cómo Generar las Hojas de Cableado

Una vez realizado todo el esquemático podemos generar las hojas de cableado y las hojas de puentes de forma automática, a partir del netlist que genera Draft Cable.

Pulsar el botón .

Aparecerá entonces el cuadro de selección de plantillas de cableado.

Las plantillas son de dos tipos:

þ     MS Word

þ     HTML

Las plantillas HTML son más rápidas, sin embargo, el usuario puede tener un mayor control sobre el formato de salida con las plantillas MS Word.
