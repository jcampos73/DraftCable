Privilegios

Draf Cable soporta varios nieveles de prvilegios. Cada uno con unas capacidades diferentes.

Siendo el nivel con más funcionalidades y permisos el de Administrador.

Administrador

Los usuarios de este nivel tienen todos los privilegios:

þ     Modificar / Eliminar planos creados por otros usuarios.

þ     Crear / Modificar / Eliminar componentes de las librerias.

þ     Ejecutar la aplicación de configuración del sistema.

Usuario avanzado

þ     Modificar / Eliminar planos creados por él mismo.

þ     Crear / Modificar / Eliminar componentes de las librerias. Sólo puede Modificar / Eliminar los componentes creados por el mismo.

Usuarios

þ     Modificar / Eliminar planos creados por él mismo.

Invitados

þ     Sólo tienen permiso de lectura para todos los objetos del sistema. No pueden crear planos.
