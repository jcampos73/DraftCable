Editor de Planos

El Editor de Planos permite el diseño gráfico del esquematico mediante diferentes herramientas seleccionables mediante comandos de las barras de herramientas y/o de los menús.

El capítulo del Editor de Planos explica en detalle todas las maniobras que se pueden realizar.

La herramienta seleccionada se indica mediante el cursor del ratón:

Cursor de Selección.

Cursor de inserción de componentes.

Cursor para aumentar el zoom.

Cursor para reducir el zoom.

Las herramientas disponibles son las siguientes:

 

Scroll

El 'Scroll' se realiza mediante las barras de desplazamiento horizontal y vertical, que también están asociadas a la rueda del ratón.

Zoom

El Zoom se realiza mediante varias herramientas situadas en la Barra de Dibujo, como son: Zoom +, Zoom-, Zoom Cursor, Zoom a un Área, Zoom a Todo.

Selección

Permite se seleccionar componentes uno a uno o mediante un área.

Inserción

Permite la inserción de todos los elementos gráficos, como: Componentes, Cable, Buses, Puertos.
