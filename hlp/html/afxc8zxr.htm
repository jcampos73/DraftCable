Comando Deshacer/No se puede deshacer (menú Editar)

Use este comando para deshacer la última acción de edición, si es posible. El nombre del comando cambia según la última acción efectuada. En el menú, el comando Deshacer cambia a No se puede deshacer, si es imposible revertir la última acción efectuada.
