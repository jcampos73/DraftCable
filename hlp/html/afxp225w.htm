Comando Imprimir (menú Archivo)

Use este comando para imprimir un documento. Este comando presenta el cuadro de diálogo Imprimir, en el cual se pueden especificar las páginas que se imprimirán, la cantidad de copias, la impresora de destino y otras opciones de configuración de impresoras.

Método abreviado

Barra de herramientas:

Teclas:CTRL+I
